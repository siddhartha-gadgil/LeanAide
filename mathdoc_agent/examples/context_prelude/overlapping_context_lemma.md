# Overlapping Local Context Lemma

## Theorem

Let \(G\) be a group and let \(l : G \to \mathbb{R}\). Assume that \(l(g)=0\) for every \(g\in G\). Then \(l(1)=0\).

## Proof

Let \(h_0\) denote the assumption that \(l(g)=0\) for every \(g\in G\).

We first prove the auxiliary lemma that, for every \(x\in G\), \(l(x)=0\). This lemma uses the same \(G\), group structure, \(l\), and \(h_0\) already fixed in the theorem context.

Applying this auxiliary lemma to \(x=1\), we get \(l(1)=0\).
