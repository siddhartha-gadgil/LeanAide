# Definition Uses Previous Definition

## Definition 1

Let \(G\) be a group. A function \(l : G \to \mathbb{R}\) is a toy pseudo-length if \(l(1)=0\).

## Definition 2

Let \(G\) be a group. A toy pseudo-length \(l : G \to \mathbb{R}\) is a toy bounded length if \(l(g)\le 1\) for every \(g\in G\).
