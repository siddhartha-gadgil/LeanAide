# Local Context and Prelude Fixture

## Definition 1

Let \(G\) be a group. A function \(l : G \to \mathbb{R}\) is a toy pseudo-length if \(l(1)=0\).

## Definition 2

Let \(G\) be a group. A toy pseudo-length \(l : G \to \mathbb{R}\) is a toy length if \(l(g)=0\) implies \(g=1\) for every \(g\in G\).

## Theorem

Let \(G\) be a group and let \(l : G \to \mathbb{R}\) be a toy length. Assume that \(l(g)=0\) for every \(g\in G\). Then \(l(1)=0\).

## Proof

Let \(h_0\) denote the assumption that \(l(g)=0\) for every \(g\in G\).

We first prove the auxiliary lemma that, for every \(x\in G\), \(l(x)=0\). This is just \(h_0\), applied to \(x\).

Applying this auxiliary lemma to \(x=1\), we get \(l(1)=0\).
